### Scaffold - 기본적인 뼈대를 잡아주는 요소 

| 속성                     | 설명                    |
| ---------------------- | --------------------- |
| `appBar`               | 상단 바 (타이틀, 뒤로가기 버튼 등) |
| `body`                 | 본문 내용 (화면의 메인 부분)     |
| `bottomNavigationBar`  | 하단 탭바 (화면 전환을 위한 버튼들) |
| `floatingActionButton` | 화면 우하단의 동그란 버튼        |
| `drawer`               | 좌측에서 나오는 슬라이드 메뉴      |
| `backgroundColor`      | 배경 색상 설정              |
