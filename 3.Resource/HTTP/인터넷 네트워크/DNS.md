
#### IP의 단점 
-  기억하기가 어렵다 
-  변경이 가능하다.

#### 이 IP를 기억하지 않고 편리하게 사용하기 위해 DNS가 등장했다.

DNS서버에 

| 도메인명       | IP            |
| ---------- | ------------- |
| google.com | 200.200.200.2 |
| aaa.com    | 210.210.210.3 |


1.클라이언트는 도메인 서버에 google.com 으로 요청
2.DNS서버에서 클라이언트 서버로 **응답**
3.클라이언트는 응답받는 IP 주소로 **접속**
