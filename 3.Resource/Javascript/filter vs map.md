
### `filter`와 `map`을 함께 사용할 때

`filter`와 `map`을 함께 사용하면 다음과 같은 효과를 얻을 수 있습니다:

1. **먼저 필터링**: `filter`로 조건에 맞는 요소를 선택합니다.
2. **다음 변환**: 그 다음, `map`으로 선택된 요소들을 변환합니다 (mapdms )                                                        
   
   
----

  

find -> 배열이 아닌 개별 객체로 list에 추가된다.
filter -> array - 배열형태로 값을 반환한다.
