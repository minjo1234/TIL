
- 특정 도메인(서비스)들의  로깅, 트랜잭션 처리, 보안 확인처럼 여러 클래스에 흩어져 있는 중복 코드를 한곳(Aspect)에 모아서 정의하고, 필요한 시점에 자동으로 끼워넣는다.
- 비즈니스 로직에만 집중하고 싶은 경우에 불필요한 로직이 서비스 Layer에 생성되지 않게 설정한다.


포인트컷 : 특정 패키지, 클래스, 어노테이이 붙은 메서드만 골라내서 부가 기능을 주입한다.
@AfterReturning : 성공적으로 메서드가 실행된 직후
