
## mount

- mount()는 Vue 객체에 컴포넌트 인스턴스를 부착하는 기낭이다.
- DOM(Document Object Model) 의 기능은 HTML을 객체처럼 사용한다.
- DOM을 Vue에서는 컴포넌트 인스턴스라고부른다.

- App.vue라는 Vue에 DOM(HTML을 객체처럼 사용하는데 Vue에서는 컴포넌트 인스턴스라고 부름)을 부착하는 것이다.

- 하나의 화면을 기준으로 내용만 변경된다.
